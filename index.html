INSTRUCTIONS FOR USE (for OS X users)

1) Download above Zip File and copy&paste the folder onto your desktop. Rename the folder to "vocab". Edit the words.txt file with the words you would like to generate mnemonics for. The words must be one per line. 2) Open Terminal and type:

cd ~/Desktop/vocab && ls

You should see three files in the output (define.py words.txt and README.txt)

3) Type into terminal:

python define.py

The program will prompt you to enter a file name. This will be the output file, so name it anything besides "words." I like to name it: "output."

4)

IF THERE ARE NO ERRORS:

Open the output.txt (or whatever you named it) and you are done.
